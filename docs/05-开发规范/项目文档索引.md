# 实验室管理系统 - 项目文档索引

> 本索引文件汇总了"基于微服务架构的实验室管理系统"项目中所有文档的路径和简要说明，方便开发人员和AI子Agent快速查阅所需文档。

---

## 文档目录总览

| 目录 | 说明 |
|------|------|
| `01-规划文档/` | 项目启动阶段产生的规划类文档，包括可行性分析、需求说明和开发计划 |
| `02-设计文档/` | 系统设计阶段产生的设计类文档，包括概要设计、详细设计、API规范和数据库设计 |
| `03-运维文档/` | 项目运维阶段需要的文档，包括测试计划、测试报告和用户操作手册 |
| `04-收尾文档/` | 项目收尾阶段产生的文档，包括进度月报、维护手册、问题报告和项目总结 |
| `05-开发规范/` | 开发过程中的规范类文档，包括编码规范、开发报告模板、Git版本控制规范和本文档索引 |
| `process/` | 各Agent开发过程中的过程文档，包括功能开发记录、变更日志等，按Agent编号分子目录存放 |

## 一、规划文档（01-规划文档）

| 序号 | 文档名称 | 文件路径 | 简要说明 |

| 序号 | 文档名称 | 文件路径 | 简要说明 |

## 文档使用指引

### 按角色查阅

| 角色 | 推荐优先阅读文档 |
|------|----------------|
| 项目管理者 | 可行性分析报告、项目开发计划、开发进度月报、项目开发总结报告 |
| 架构师 | 概要设计说明书、API接口设计规范、数据库设计说明书、编码规范 |
| 后端开发（AI子Agent） | 软件需求说明书、详细设计说明书、API接口设计规范、数据库设计说明书、编码规范（后端部分）、开发报告模板 |
| 前端开发（AI子Agent） | 软件需求说明书、详细设计说明书、API接口设计规范、编码规范（前端部分）、开发报告模板 |
| 测试人员 | 软件需求说明书、测试计划、测试分析报告、用户操作手册 |
| 运维人员 | 软件维护手册、Docker部署规范（编码规范中）、软件问题报告 |

### 按开发阶段查阅

| 开发阶段 | 推荐阅读文档 |
|---------|-------------|
| 项目启动 | 可行性分析报告、项目开发计划 |

| 系统设计 | 概要设计说明书、详细设计说明书、API接口设计规范、数据库设计说明书 |
| 编码开发 | 编码规范、开发报告模板 |
| 测试验证 | 测试计划、测试分析报告 |
| 部署上线 | 软件维护手册、编码规范（Docker部署部分） |
| 项目收尾 | 项目开发总结报告、软件问题报告 |
